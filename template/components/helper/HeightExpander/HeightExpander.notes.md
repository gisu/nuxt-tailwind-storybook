## HeightExpander
Animates the height of an element that is opened and closed with v-if 

### Props
- `timing [string]`: Timing der Animation [fast/slow/moderate]

### Code
```jsx
<HeightExpander>
  <div v-if="test">
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Amet aspernatur cum cumque, labore obcaecati quasi quidem! Assumenda dignissimos doloremque explicabo illo, iste laborum magni neque, omnis reiciendis saepe, vel voluptatem.</p>
  </div>
</HeightExpander>
```
